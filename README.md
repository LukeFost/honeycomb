# 🍯 honeycomb

pnpm + Turborepo monorepo. ETHGlobal NY hackathon.

## Stack

- **pnpm** workspaces + **Turborepo**
- `apps/web` — Next.js 16 (App Router, TypeScript, Tailwind)
- `packages/*` — shared code (empty for now)

## Get started

```bash
pnpm install
pnpm dev          # runs apps/web on http://localhost:3000
```

Other commands: `pnpm build`, `pnpm lint`, `pnpm start`.
